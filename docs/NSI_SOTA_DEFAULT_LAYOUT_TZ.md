# ТЗ: дефолтная планировка офиса СОТА

## 1. Цель

Настроить стартовую планировку левого крыла офиса СОТА на 4 этаже так, чтобы она выглядела как офисная схема, а не как случайная сетка плиток.

## 2. Требуемая композиция

Внутри контейнера `Офис СОТА, левое крыло`:

- кабинеты должны стоять по углам офиса;
- в центре должны быть переговорки и кухня;
- open space должны идти по периметру сверху и снизу;
- open space должны чередоваться с кабинетами;
- серверная и коридор остаются внутри офиса и не должны конфликтовать с основными зонами.

## 3. Ограничения

- Это дефолтная стартовая расстановка.
- Пользователь может дальше редактировать планировку вручную.
- Координаты задаются в процентах от поля планировки.
- Родительский контейнер `Офис СОТА, левое крыло` остается фоном для дочерних помещений.

## 4. Критерии приемки

1. На 4 этаже офис СОТА отображается как большой контейнер левого крыла.
2. 4 кабинета стоят по углам офиса.
3. Переговорки и кухня расположены ближе к центру.
4. Open space расположены сверху и снизу по периметру.
5. Open space чередуются с кабинетами.
6. Редактирование планировки продолжает работать.
