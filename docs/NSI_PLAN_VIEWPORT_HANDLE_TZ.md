# ТЗ: изменение окна планировки через уголок

## 1. Цель

Упростить настройку окна просмотра планировки.

Вместо текстовых настроек ширины/высоты и кнопок `+ окно`, `- окно`, `Широкий БЦ` нужно оставить простой визуальный способ изменения размера: уголок снизу справа.

## 2. Требования

В режиме `Планировка`:

- убрать панель настройки окна просмотра;
- убрать панель настройки поля планировки;
- оставить кнопку `В центр`;
- оставить кнопку `Редактировать`;
- добавить уголок снизу справа у окна просмотра;
- при перетаскивании уголка меняется размер видимой области планировки;
- если окно шире рабочей области интерфейса, остается горизонтальная прокрутка.

## 3. Ограничения

- Размер внутреннего поля планировки не должен меняться автоматически при изменении окна просмотра.
- Координаты плиток не должны меняться.
- Редактор плиток должен продолжать работать.
- Перемещение самой канвы должно продолжать работать.

## 4. Критерии приемки

1. Панели с числовыми настройками окна и поля не отображаются.
2. У окна планировки есть уголок снизу справа.
3. Уголок можно тянуть мышью.
4. Видимое окно становится шире/уже и выше/ниже.
5. Планировка не ломается и не пересчитывает координаты помещений.
