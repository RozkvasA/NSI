# ТЗ: стартовая планировка СОТА по пользовательскому чертежу

## 1. Цель

Настроить стартовый экран НСИ так, чтобы при открытии сразу отображалась планировка 4 этажа, а не дерево объектов.

Планировка офиса СОТА должна быть переложена по пользовательскому чертежу и описанию рядов помещений.

## 2. Стартовый экран

При открытии модуля:

- активна вкладка `Объекты и помещения`;
- активен режим `Планировка`;
- выбран 4 этаж;
- выбран объект `Офис СОТА, левое крыло`.

## 3. Общая структура офиса

Офис вытянут горизонтально.

Слева расположен вход в офис.

Структура сверху вниз:

1. Верхний ряд помещений.
2. Верхний коридор.
3. Центральный блок помещений.
4. Нижний коридор.
5. Нижний ряд помещений.

## 7. Критерии приемки

1. Модуль открывается сразу на планировке.
2. На 4 этаже отображается вытянутый офис.
3. Верхний ряд соответствует описанию слева направо.
4. Центральный блок соответствует описанию слева направо.
5. Нижний ряд соответствует описанию слева направо.
6. У помещений указаны площади.
7. Коридоры отделяют верхний, центральный и нижний уровни.
8. Редактирование, масштаб и расширение белой области продолжают работать.
