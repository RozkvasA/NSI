# ТЗ: изменение размера поля планировки

## 1. Цель

Добавить возможность менять размер самого поля планировки.

Пользователь должен управлять не только плитками помещений, но и размером рабочей канвы, внутри которой эти плитки расположены.

## 2. Основное поведение

В режиме `Планировка` должны появиться элементы управления размером поля:

- ширина поля;
- высота поля;
- уменьшить поле;
- увеличить поле;
- сбросить размер.

## 3. Правило координат

Координаты помещений остаются в процентах от поля планировки.

При изменении размера поля:

- относительное расположение помещений сохраняется;
- визуальный размер помещений меняется вместе с полем;
- редактирование плиток продолжает работать.

## 4. Ограничения

Не требуется:

- backend-сохранение;
- сложное масштабирование с колесом мыши;
- бесконечная канва;
- CAD-редактор.

## 5. Критерии приемки

1. Размер поля планировки можно менять.
2. Есть поля ввода ширины и высоты.
3. Есть быстрые кнопки увеличения и уменьшения.
4. Есть сброс размера.
5. Перемещение поля продолжает работать.
6. Редактирование плиток продолжает работать.
