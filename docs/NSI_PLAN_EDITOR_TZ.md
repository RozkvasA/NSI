# ТЗ: редактирование планировки этажей

## 1. Цель

Добавить в режим `Планировка` возможность редактировать расположение, размер и цвет плиток помещений.

Планировка должна оставаться простой схемой для навигации, но пользователь должен иметь возможность вручную привести этаж к нужному виду без изменения кода.

## 2. Основное поведение

В режиме `Планировка` должна появиться кнопка `Редактировать`.

После включения режима редактирования пользователь может:

- выбрать плитку помещения;
- изменить положение плитки на плане;
- изменить ширину и высоту плитки;
- изменить цвет плитки;
- сохранить изменения;
- отменить несохраненные изменения.

## 3. Редактирование положения

Плитку можно перемещать по плану drag-and-drop.

Требования:

- плитка не должна выходить за границы планировки;
- положение хранится в процентах от ширины и высоты плана;
- после перемещения карточка помещения справа должна оставаться связанной с этой же сущностью.

## 4. Редактирование размера

Для выбранной плитки должны быть доступны поля:

- X;
- Y;
- ширина;
- высота.

Значения задаются в процентах.

Также у выбранной плитки должен быть визуальный маркер изменения размера в правом нижнем углу.

## 5. Редактирование цвета

Для выбранной плитки должен быть доступен выбор цвета.

Цвет используется только для визуального оформления плитки на плане и не должен менять статус помещения.

## 6. Сохранение

Изменения должны сохраняться в текущем состоянии приложения.

В рамках статического прототипа сохранение выполняется в памяти браузера до перезагрузки страницы.

## 7. Отмена

Кнопка `Отменить` должна вернуть планировку к состоянию на момент включения режима редактирования.

## 8. Ограничения

Не требуется:

- backend;
- постоянное сохранение в базу;
- CAD-редактор;
- загрузка чертежей;
- сложные слои и координатные сетки.

## 9. Критерии приемки

Задача считается выполненной, если:

1. В режиме `Планировка` есть кнопка `Редактировать`.
2. В режиме редактирования можно выбрать плитку помещения.
3. Выбранную плитку можно переместить мышью.
4. Можно изменить X, Y, ширину и высоту через поля.
5. Можно изменить цвет плитки.
6. Можно сохранить изменения.
7. Можно отменить изменения.
8. После редактирования плитка остается кликабельной и открывает карточку помещения.
