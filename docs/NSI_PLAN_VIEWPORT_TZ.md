# ТЗ: настройка окна просмотра планировки

## 1. Цель

Добавить настройку размера окна просмотра планировки, чтобы пользователь мог видеть не только левое крыло, но и полноценное правое крыло вытянутого этажа.

Важно разделить два понятия:

- `поле планировки` — внутренняя канва, где лежат координаты помещений;
- `окно просмотра` — видимая область, через которую пользователь смотрит на поле планировки.

## 2. Требования

В режиме `Планировка` должны появиться настройки окна просмотра:

- ширина окна;
- высота окна;
- уменьшить окно;
- увеличить окно;
- широкий режим;
- сброс окна.

## 3. Поведение

При изменении размера окна:

- координаты помещений не меняются;
- размер поля планировки не меняется;
- меняется только видимая область просмотра;
- если окно уже экрана, должна быть горизонтальная прокрутка;
- правое крыло должно быть доступно без изменения координат помещений.

## 4. Совместимость

Доработка не должна ломать:

- перемещение поля планировки;
- изменение размера поля планировки;
- редактирование плиток;
- вертикальные подписи узких плиток;
- предупреждение при выходе помещения из родителя.

## 5. Критерии приемки

1. В планировке есть настройки окна просмотра.
2. Можно задать ширину и высоту окна вручную.
3. Можно быстро увеличить или уменьшить окно.
4. Есть широкий режим для вытянутого БЦ.
5. Есть сброс размера окна.
6. При широком окне помещается больше правого крыла.
7. При нехватке места появляется прокрутка, а не обрезание схемы.
