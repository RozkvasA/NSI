# НСИ

Статический прототип модуля НСИ для ведения структуры объекта, помещений, систем, оборудования, техкарт и справочников.

## Что внутри

- вкладки без декоративных символов:
  - Объекты и помещения;
  - Системы и оборудование;
  - Техкарты;
  - Справочники;
- отдельная пространственная структура объекта;
- отдельная иерархия систем и оборудования;
- режимы `Дерево` и `Планировка` для помещений;
- карточка помещения с прямыми, наследуемыми и нижележащими системами;
- карточка помещения показывает конкретные единицы оборудования, а не группы как физические строки;
- карточки систем, оборудования и техкарт;
- создание объектов, систем, оборудования, ТК и значений справочников;
- массовое создание объектов;
- drag-and-drop перемещение объектов в дереве;
- действие `Изменить родителя` в карточке объекта;
- демо-структура БЦ Lucky / Офис СОТА.

## Запуск

Открыть `index.html` в браузере или запустить локальный сервер:

```bash
python -m http.server 8080
```

После запуска открыть:

```text
http://localhost:8080
```
