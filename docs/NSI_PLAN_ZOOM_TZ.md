# ТЗ: масштабирование окна планировки

## 1. Цель

Добавить отдельное масштабирование планировки внутри окна просмотра.

Масштабирование должно быть независимым от:

- размера окна просмотра;
- размера белой области планировки;
- размеров и координат помещений в данных.

## 2. Требования

В режиме `Планировка`:

- колесо мыши над планировкой меняет масштаб;
- маленькая кнопка `100%` сбрасывает масштаб;
- масштаб отображается компактным бейджем;
- текстовые панели настроек не возвращаются;
- внешний уголок продолжает менять окно просмотра;
- внутренний уголок продолжает расширять белую область;
- плитки можно переносить в расширенную часть белой области.

## 3. Поведение при масштабировании

При изменении масштаба:

- визуально масштабируется белая область с плитками;
- координаты плиток в данных не меняются;
- размер белой области в данных не меняется;
- размер окна просмотра не меняется.

## 4. Совместимость drag/resize

Перемещение и изменение размера плиток должны учитывать текущий масштаб, чтобы движение мыши корректно переводилось в координаты планировки.

## 5. Критерии приемки

1. План можно приближать и отдалять колесом мыши.
2. Есть компактный сброс масштаба до 100%.
3. Масштабирование не меняет координаты помещений.
4. После расширения белой области помещения можно перетаскивать в новую зону.
5. После масштабирования перетаскивание помещений остается корректным.
